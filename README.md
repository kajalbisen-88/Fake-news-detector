# Fake News Detector

A web app that detects whether a news headline is **fake** or **real** using machine learning (TF-IDF + Logistic Regression / SVM) and built with **Streamlit**.

---

## Features

- Predict if a news headline is fake or real.
- User-friendly web interface using Streamlit.
- Batch upload for multiple headlines via CSV.
- Shows prediction confidence.
